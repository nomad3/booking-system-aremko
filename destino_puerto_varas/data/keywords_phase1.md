# DPV-SEO-002 — Fase 1 Keyword Research

**Fecha:** 2026-04-30
**Sitio objetivo:** www.destinopuertovaras.cl (DPV)
**Stack usado:** Google Suggest API (público), conocimiento de dominio
**Seeds investigados:** 17 queries semilla
**Output:** 45 keywords priorizadas en 9 clusters

> **Alcance Fase 1:** este trabajo es 100% para **destinopuertovaras.cl**, posicionado como medio editorial neutral del destino Puerto Varas. **No mezcla con aremko.cl** (sitio de marca, etapa posterior). Cada keyword/post se evalúa por su valor para DPV como guía del destino, no por fit con un proveedor específico.

> **Origen:** sesión Fase 1 ejecutada con stack gratuito tras rechazo de Semrush (decisión #4 lockeada). Volúmenes son **estimaciones cualitativas** (Alto/Medio/Bajo) basadas en frecuencia en autocomplete + intent. Para validar volúmenes exactos antes de publicar, usar Ubersuggest/AnswerThePublic puntualmente sobre las P0.

## Leyenda

| Campo | Valores |
|---|---|
| **Cluster** | `GUIDES`, `ITINERARIES`, `HOWTOS`, `SEASONS`, `FAMILY`, `ROMANCE`, `GASTRONOMY`, `CULTURE`, `NATURE` |
| **Intent** | `INFO` (informacional), `NAV` (navegacional), `COMM` (comercial/comparativa), `TXN` (transaccional) |
| **Vol** | `A` alto · `M` medio · `B` bajo (relativo Chile/español) |
| **Prio** | `P0` primer batch (top 5-8) · `P1` siguientes 3-6 meses · `P2` backlog |
| **Fit DPV** | qué tan bien la keyword apoya autoridad editorial del destino y conecta con circuitos / places del catálogo DPV |

## Filtros aplicados

Excluidas del set por no aportar a estrategia:
- `puerto varas tiempo / clima / temperatura` → Google sirve widget directo, sin oportunidad de blog
- `puerto varas municipalidad / clinica / permiso de circulacion` → fuera de tema turismo
- `puerto varas a puerto montt` (sin intent turístico claro)
- `lago llanquihue contaminado / profundidad / letras` → curiosidad sin fit comercial
- `volcan osorno erupcion / activo / altura` → curiosidad geológica
- Variantes en portugués (`roteiro`) → idioma no foco Fase 1
- Top-of-funnel comercial inmediato (`arriendo`, `pasajes`, `cabañas`) → no es blog informacional
